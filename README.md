
# 영단어 카드짝 맞추기 게임 만들기

## 소개
### 1. 영어단어와 그 단어의 의미를 가진 한국어의 짝을 맞추어 카드를 모두 뒤집으면 클리어 하는 간단한 게임
### 2. 현재는 8세트를 기준으로 카드를 생성할 수 있다.

## URL 
## https://flipword-beta.netlify.app/

## Tech
### 1. React 기반 개발
### 2. Material UI(MUI), styled-components 등 기타 스타일 관련 라이브러리 사용
### 3. react-toastify로 알림창 표시
